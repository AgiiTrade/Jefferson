QuestMind Support

Support page for the QuestMind mobile app and web experience.

Need help?

If you run into a bug, loading issue, or account/device problem, send the following in your message:

  • device type and OS version
  • what screen you were on
  • what happened vs what you expected
  • a screenshot if possible

Email: support@agii.ca

Common issues

  • If the game seems stuck, close and reopen the app.
  • If visuals look outdated, make sure you are on the latest version.
  • If audio or animations feel off, restart the app after updating.
