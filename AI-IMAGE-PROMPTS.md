## 🖼️ HOW TO GENERATE

### Option 1: Midjourney (Best Quality)
1. Go to midjourney.com or Discord
2. Type: `/imagine [paste prompt]`
3. Add: `--ar 8.5:11 --style raw`
4. Generate 4 variations
5. Upscale best one
6. Download PNG

### Option 2: DALL-E (Easiest)
1. Go to chat.openai.com
2. Paste prompt
3. Generate
4. Download PNG

### Option 3: Leonardo AI (Free)
1. Go to leonardo.ai
2. Select "Line Art" preset
3. Paste prompt
4. Generate
5. Download PNG

## 🐍 PYTHON ASSEMBLY SCRIPT

```python
# Save this as assemble.py and run: python3 assemble.py
from PIL import Image
import os

def images_to_pdf(image_folder, output_pdf):
    images = []
    for filename in sorted(os.listdir(image_folder)):
        if filename.endswith('.png'):
            img = Image.open(os.path.join(image_folder, filename))
            img = img.convert('RGB')
            images.append(img)
    
    if images:
        images[0].save(output_pdf, save_all=True, append_images=images[1:])
        print(f"Created: {output_pdf}")

# Usage:
# images_to_pdf('dinosaur-images/', 'dinosaur-adventures.pdf')
```
