# Agii Mortgage Decisioning Copilot

A separate product concept and app shell for the Document AI / mortgage underwriting workflow discussed with Arfeen.

## Product intent
This is not AI Modernizer. This is a separate lending-focused product that combines:
- document intake
- OCR / classification / extraction
- structured borrower profile generation
- financial ratio calculations
- regulatory and underwriting rule checks
- risk scoring
- explainable decision support
- underwriter enablement
- on-prem / private deployment posture

## Current build
Initial interactive prototype in `index.html` with:
- decision summary
- document completeness and traceability
- weighted risk factor view
- rule trace / explainability layer
- on-prem security posture section
