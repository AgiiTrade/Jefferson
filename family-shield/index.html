📱 Device Setup Instructions

iPhone/iPad:

  1. Settings → Screen Time → Turn On
  2. Set up Family Sharing
  3. Add child's Apple ID
  4. Enable Content & Privacy Restrictions

Android:

  1. Install Google Family Link
  2. Create supervised Google Account
  3. Link child's device
  4. Set screen time limits

Windows PC:

  1. Settings → Accounts → Family
  2. Add child's Microsoft account
  3. Enable activity reporting
