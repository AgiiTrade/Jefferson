# 🛡️ FamilyShield Chrome Extension — Installation Guide

## 📥 How to Install

### Step 1: Download Extension
1. Download the `extension` folder from GitHub
2. Or clone the repo: `git clone https://github.com/AgiiTrade/Jefferson.git`

### Step 2: Enable Developer Mode
1. Open Chrome
2. Go to: `chrome://extensions/`
3. Turn ON **"Developer mode"** (top right toggle)

### Step 3: Load Extension
1. Click **"Load unpacked"**
2. Select the `family-shield/extension` folder
3. Extension will appear in your toolbar

### Step 4: Pin the Extension
1. Click the **puzzle piece icon** 🧩 in toolbar
2. Find **"FamilyShield"**
3. Click the **pin icon** 📌

## 🔧 Setup

### Default PIN
- **Default PIN:** `1234`
- **Change it immediately** after installation

### Configure for Your Child
1. Click the FamilyShield icon
2. Enter PIN: `1234`
3. Go to **Settings** tab
4. Set daily screen time limit
5. Change PIN to something only you know
6. Lock the extension

## ✅ Features

| Feature | Description |
|---------|-------------|
| **Screen Time Tracking** | Tracks time spent on each website |
| **Daily Limits** | Set max hours per day |
| **Site Blocking** | Blocks adult/gambling/violent sites |
| **Time Warnings** | Alerts at 80% and 100% of limit |
| **Activity Reports** | See top sites visited |
| **PIN Protection** | Settings locked behind PIN |
| **Dashboard Integration** | Syncs with FamilyShield web dashboard |

## 📱 Mobile Setup (iOS/Android)

### iPhone/iPad:
1. Settings → Screen Time → Turn On
2. Tap "Use Screen Time Passcode" → Set a passcode
3. Go to Content & Privacy Restrictions → Enable
4. Set Web Content → Limit Adult Websites
5. Set App Limits for specific apps

### Android:
1. Install **Google Family Link** from Play Store
2. Create a supervised Google Account for your child
3. Link their device to your Family Link
4. Set daily limits and content filters

## 🔒 Security Tips

1. **Change the default PIN** immediately
2. **Don't share the PIN** with your child
3. **Lock the extension** when not configuring
4. **Check reports** weekly
5. **Update blocklist** as needed

## 🆘 Troubleshooting

**Extension not blocking sites?**
- Make sure Developer Mode is ON
- Reload the extension
- Check blocklist.json is loaded

**Time not tracking?**
- Grant all permissions when prompted
- Reload Chrome
- Check background.js is running

**Can't unlock?**
- Default PIN is `1234`
- If forgotten, uninstall and reinstall extension
