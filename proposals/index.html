✏️ How to Customize

Quick Start Guide

  1. Open the template in any browser or code editor
  2. Find [PLACEHOLDER] text — search for brackets to find all customizable fields
  3. Replace placeholders with client-specific details: company name, project scope, pricing
  4. Adjust pricing — each proposal has tiered pricing tables. Edit the amounts to match your quote
  5. Update case studies — replace generic examples with real project outcomes
  6. Add your team — update names, roles, and bios in the Team section
  7. Print to PDF — use Ctrl+P / Cmd+P for a clean print-friendly version
  8. Use the Pricing Calculator for quick estimates, then copy values into the proposal
