# 📖 Setup Guide — How to Import Templates into Notion

## Quick Start

### Option 1: Duplicate from Template Link (Recommended)

1. **Click the template link** provided in your purchase email
2. Click the **"Duplicate"** button in the top-right corner
3. The template will be added to your Notion workspace
4. Navigate to it in your sidebar and start customizing

### Option 2: Import from Markdown Files

1. Open Notion and go to the workspace where you want the template
2. Click **⋯** (three dots) in the top-right corner
3. Select **"Import"** → **"Markdown & CSV"**
4. Navigate to the template `.md` file on your computer
5. Click **"Open"** — Notion will import it with formatting intact

### Option 3: Manual Setup (Step-by-Step)

If you prefer to build from scratch using our detailed instructions:

1. Open the template description file (e.g., `01-CRM-TEMPLATE.md`)
2. Follow the **"Setup Instructions"** section exactly
3. Create each database property as described
4. Add the example data to test your setup

## Step-by-Step Setup for Each Template

### 1. Create Your Template Hub

First, create a central page to organize all templates:

1. Open Notion → Click **"+ New Page"** in the sidebar
2. Name it **"My Templates"** or **"Ultimate Templates Bundle"**
3. Add a divider and description

### 2. Import Each Template

For each of the 15 templates:

1. Create a new sub-page under your hub
2. Name it according to the template (e.g., "CRM Dashboard")
3. Import the corresponding `.md` file
4. Or follow manual setup instructions

### 3. Configure Database Properties

Each template uses Notion databases. Here's how properties work:

**Creating a Database:**
- Type `/database` and select "Table - Inline"
- Click **"+ Add a property"** to add columns
- Choose property types: Text, Number, Select, Date, etc.

**Property Types Used:**
| Type | Use For |
|------|---------|
| Title | Primary identifier (auto-created) |
| Text | Names, descriptions, notes |
| Number | Amounts, counts, percentages |
| Select | Single-choice statuses |
| Multi-Select | Multiple tags/categories |
| Date | Dates and deadlines |
| Checkbox | Yes/No toggles |
| Person | Team member assignment |
| Files & Media | Attachments, images |
| URL | Web links |
| Formula | Calculated values |
| Relation | Link databases together |
| Rollup | Aggregate data from relations |

### 4. Create Views

Most templates include multiple views. To add views:

1. Click **"+ Add a view"** at the top-left of any database
2. Choose: Table, Board, Calendar, Gallery, List, Timeline
3. Configure: Filter, Sort, Group by specific properties
4. Rename each view for clarity

### 5. Customize for Your Needs

**Common Customizations:**
- Change Select/Tag colors to match your brand
- Add your own categories and statuses
- Adjust formulas for your specific calculations
- Create filtered views for focused work

## Troubleshooting

### Template Didn't Import Correctly?

1. **Check file format** — Make sure it's `.md` (Markdown)
2. **Try re-importing** — Sometimes a second attempt works
3. **Manual setup** — Follow the detailed instructions in each template file
4. **Check Notion version** — Update to the latest Notion app

### Properties Not Showing?

1. Some properties require specific types (e.g., Formula needs inputs first)
2. Relation properties need both databases to exist
3. Rollup properties need a Relation property to reference

### Views Not Working?

1. Make sure you're grouping by the right property
2. Check that filters aren't hiding all your data
3. Ensure date properties are actual Date types, not Text

## Best Practices

### 🎯 Start Simple
Don't customize everything at once. Use the templates as-is for a week, then adjust.

### 🔗 Link Templates Together
Use Relations to connect:
- CRM → Invoice Tracker (link clients to invoices)
- Meeting Notes → Project Manager (link meetings to projects)
- Goal Setting → Weekly Review (track OKR progress)

### 📱 Use Notion Mobile
All templates work on Notion's mobile app. Great for:
- Quick habit check-ins
- Logging workouts
- Capturing journal entries
- Adding book notes on the go

### 🔄 Review Regularly
- **Daily:** Check your task lists and habit tracker
- **Weekly:** Complete the Weekly Review template
- **Monthly:** Review goals and financial tracking
- **Quarterly:** OKR check-in and retrospective

### 💾 Backup Your Data
Notion doesn't have automatic backups. Export regularly:
1. Click **⋯** → **Export** → **"Markdown & CSV"**
2. Save to cloud storage (Google Drive, Dropbox)
