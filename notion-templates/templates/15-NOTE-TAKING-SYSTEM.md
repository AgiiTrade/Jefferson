# 🧠 Note-Taking System (Zettelkasten)

## Description

A powerful note-taking system based on the Zettelkasten method. Create atomic notes, build connections between ideas, and develop a personal knowledge graph that compounds in value over time.

## Features

- **Atomic Notes** — One idea per note, written in your own words
- **Permanent Notes** — Distilled insights from fleeting and literature notes
- **Bi-directional Links** — Connect related notes with backlinks
- **Topic Categories** — Organize by area of knowledge
- **Index Notes** — Entry points into topic clusters
- **Daily Notes** — Quick capture for processing later
- **Evergreen Notes** — Continuously refined over time

## Setup Instructions

1. Create a new page: "Note-Taking System"
2. Create **Permanent Notes Database**:
   - Title (Title — clear, declarative statement)
   - Unique ID (Formula or Text — e.g., "20260310-001")
   - Content (Text — long)
   - Category (Multi-select: Business, Technology, Psychology, Philosophy, Science, Personal)
   - Related Notes (Relation to self — bidirectional links)
   - Source (Text or Relation — where the idea came from)
   - Created (Created Time)
   - Last Updated (Last Edited Time)
   - Status (Select: Fleeting, Literature, Permanent, Evergreen)
   - Tags (Multi-select)
3. Create **Literature Notes Database**:
   - Source Title (Title)
   - Author (Text)
   - Type (Select: Book, Article, Video, Podcast, Course)
   - URL (URL)
   - Key Ideas (Text — long)
   - Permanent Notes (Relation to Permanent Notes)
   - Date Read (Date)
4. Create **Index Notes** as regular Notion pages for each major topic
5. Create Table View sorted by Created (newest)
6. Create Gallery View for visual browsing

## Customization Tips

- Use **Notion's built-in @-mentions** for linking notes
- Create **Maps of Content (MOC)** for topic overviews
- Add **Importance** rating to prioritize note refinement
- Set up a **Weekly Review** workflow to process fleeting notes
- Use **Synced Blocks** to reference key ideas across pages
