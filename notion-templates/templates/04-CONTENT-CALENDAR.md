# рҹ“… Content Calendar

## Description

Plan, schedule, and manage all your content across platforms. Perfect for social media managers, bloggers, and newsletter creators who need a bird's-eye view of their content pipeline.

## Features

- **Multi-Platform Support** вҖ” Instagram, Twitter/X, LinkedIn, Blog, Newsletter, YouTube
- **Content Pipeline** вҖ” Idea вҶ’ Draft вҶ’ Review вҶ’ Scheduled вҶ’ Published
- **Calendar View** вҖ” Visual monthly/weekly content schedule
- **Content Ideas Bank** вҖ” Capture ideas anytime, schedule later
- **Performance Tracking** вҖ” Views, likes, shares, comments per post
- **Batch Planning** вҖ” Plan weeks of content in one session

## Setup Instructions

1. Create a new page: "Content Calendar"
2. Create the **Content Database**:
   - Title (Title)
   - Platform (Multi-select: Instagram, Twitter, LinkedIn, Blog, Newsletter, YouTube)
   - Status (Select: Idea, Draft, Review, Scheduled, Published)
   - Publish Date (Date)
   - Content Type (Select: Post, Story, Reel, Article, Video)
   - Caption/Body (Text)
   - Media (Files & Media)
   - Hashtags (Text)
   - Engagement (Number вҖ” for tracking metrics)
3. Create Calendar View by Publish Date
4. Create Board View grouped by Status
5. Create Table View filtered by Platform

## Customization Tips

- Add a **Campaign** relation to group content by marketing campaign
- Use **Templates** within the database for each platform's format
- Create a **Content Repurposing** workflow (blog вҶ’ social posts вҶ’ newsletter)
- Add **Approval** checkbox for team workflows
