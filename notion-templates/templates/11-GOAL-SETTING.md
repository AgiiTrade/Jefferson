# 🎯 Goal Setting (OKRs)

## Description

Set and achieve your goals using the proven OKR (Objectives & Key Results) framework. Break big dreams into quarterly objectives with measurable key results, track progress, and conduct regular reviews.

## Features

- **Annual Vision** — Big-picture goals for the year
- **Quarterly OKRs** — 3-5 objectives with 3-5 key results each
- **Progress Tracking** — Visual progress bars and percentages
- **Weekly Check-ins** — Regular progress updates
- **Confidence Score** — Self-assessed likelihood of hitting targets
- **Retrospectives** — End-of-quarter reviews and learnings

## Setup Instructions

1. Create a new page: "Goal Setting"
2. Create **Objectives Database**:
   - Objective (Title)
   - Quarter (Select: Q1, Q2, Q3, Q4)
   - Category (Select: Business, Personal, Health, Learning, Financial)
   - Status (Select: Not Started, In Progress, At Risk, Complete)
   - Progress (Number — percentage)
   - Confidence (Select: 🟢 High, 🟡 Medium, 🔴 Low)
   - Description (Text)
3. Create **Key Results Database**:
   - Key Result (Title)
   - Objective (Relation to Objectives)
   - Target Value (Number)
   - Current Value (Number)
   - Progress (Formula)
   - Status (Select: Not Started, On Track, Behind, Complete)
   - Notes (Text)
4. Create **Weekly Check-ins Database**:
   - Week (Title)
   - Progress Notes (Text)
   - Blockers (Text)
   - Next Week Focus (Text)

## Customization Tips

- Add **OKR Templates** for business vs personal goals
- Use **Rollups** to auto-calculate objective progress from key results
- Create **Vision Board** gallery view with goal images
- Add **Milestone** dates for intermediate checkpoints
