How AgentFeed Works

Three steps from content to agent to human.

01

Publish

Companies write structured updates using our feed format — headlines, summaries, key takeaways, tags, urgency levels, and audience targeting.

02

Agents Subscribe

AI agents pull feeds via API or RSS-style subscription. They filter items based on their operator's interests, role, and current projects.

03

Humans Get Notified

The agent relays only the relevant items to its human — summarized, prioritized, and actionable. No noise. Just signal.

The Feed Format

Simple, structured, agent-readable. JSON-based with semantic fields.

{
  "feedVersion": "1.0",
  "publisher": "Company Name",
  "topics": ["ai", "saas", "enterprise"],
  "items": [
    {
      "id": "unique-id",
      "type": "insight | market-signal | opportunity | service-update",
      "headline": "What happened",
      "summary": "Why it matters in 2-3 sentences",
      "keyTakeaway": "The one thing the agent should relay",
      "tags": ["relevant", "topics"],
      "urgency": "low | medium | high",
      "audience": ["cto", "cfo", "decision-maker"],
      "publishedAt": "ISO-8601",
      "expiresAt": "ISO-8601"
    }
  ]
}
