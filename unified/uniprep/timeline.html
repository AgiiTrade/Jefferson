📅 College Prep Timeline

Year-by-year roadmap to admission at top universities

🟢 Freshman Year (9th Grade) Foundation

Fall
✓ Take challenging courses (Honors if available)
✓ Join 2-3 clubs that interest you
✓ Meet your school counselor
Spring
✓ Explore extracurricular activities
✓ Start building study habits
✓ Begin thinking about potential majors
Summer
✓ Read widely in your areas of interest
✓ Consider summer programs or volunteering
✓ Start a personal project
Key Goals
✓ GPA: 3.7+ (unweighted)
✓ Establish 2-3 strong activities
✓ Develop good study habits

🔵 Sophomore Year (10th Grade) Building

Fall
✓ Take PSAT (practice)
✓ Start AP courses (1-2)
✓ Take leadership roles in activities
Spring
✓ Take SAT Subject Tests if applicable
✓ Deepen involvement in activities
✓ Begin SAT/ACT prep
Summer
✓ SAT/ACT prep course or self-study
✓ Summer program or internship
✓ Visit college campuses
Key Goals
✓ GPA: 3.8+ (unweighted)
✓ Take 2-3 AP courses
✓ SAT practice: 1300+

🟡 Junior Year (11th Grade) Critical Year

Fall
✓ Take PSAT/NMSQT (National Merit)
✓ Take 3-4 AP courses
✓ Begin SAT/ACT serious prep
Spring
✓ Take SAT/ACT (first official attempt)
✓ Build college list (reach/match/safety)
✓ Ask teachers for recommendations
Summer
✓ Retake SAT/ACT if needed
✓ Start Common App essay
✓ Visit top choice colleges
✓ Research scholarships
Key Goals
✓ GPA: 3.9+ (unweighted)
✓ SAT: 1450+ or ACT: 33+
✓ Strong leadership roles
✓ College list: 8-12 schools

🔴 Senior Year (12th Grade) Application Time

Fall (Sep-Nov)
✓ Finalize college list
✓ Complete Common App
✓ Submit Early Decision/Action apps
✓ Request recommendation letters
Winter (Dec-Feb)
✓ Submit Regular Decision apps
✓ Complete FAFSA (Oct 1)
✓ Apply for scholarships
✓ Send mid-year reports
Spring (Mar-Apr)
✓ Receive decisions
✓ Compare financial aid offers
✓ Visit accepted schools
✓ Make final decision (May 1)
Summer
✓ Send final transcripts
✓ Attend orientation
✓ Prepare for college life!
